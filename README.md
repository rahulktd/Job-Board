

# Job Board using Django

This project provides a web application for posting job listings and searching for job opportunities. It is developed using the Django web framework.

## Features

- User registration and authentication: Users can register, log in, and manage their job postings and applications.
- Job posting: Employers can create and manage job listings, including job title, description, requirements, and contact information.
- Job search: Job seekers can search for job opportunities based on various criteria such as location, keywords, and category.
- Application submission: Job seekers can apply for jobs directly through the platform by submitting their resume and contact details.
